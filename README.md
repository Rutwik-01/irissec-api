# IrisSec — Incident Response Intelligence System

A RESTful API for tracking and managing cybersecurity incidents,
built to demonstrate a complete 7-stage Jenkins DevOps pipeline.

## Tech Stack
Node.js 20, Express.js, SQLite, Docker, Jenkins, SonarQube, Prometheus, Grafana

## Pipeline Stages
Build → Test → Code Quality → Security → Deploy → Release → Monitoring

## Setup
Clone the repo, open Jenkins, create a Pipeline job pointing to this
repository, and click Build Now. See the demo video for a full walkthrough.
